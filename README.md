# TechTitans
An E- commerce website to purchase pre- build
custom high-end PCs for gaming, graphic
designing, autocad, etc purposes. You can also
buy PC parts from this website.
Frontend-HTML, CSS,JS,Bootstrap
Backend-PHP
Database- MySql
